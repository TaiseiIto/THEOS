# THEOS

My original operating system made with reference to [MikanOS](https://github.com/uchan-nos/mikanos/) by uchan-nos.

## thoes-devenv

To develop and use THEOS conveniently, I prepared [Dockerfile](/blob/main/Dockerfile).
The docker image is built by the Dockerfile with `make build-devenv` and the container is run with `make run-devenv`.

## [Makefile](/blob/main/Makefile)

The Makefile contains the following commands.

* make
* make build-devenv
* make clean
* make login-devenv
* make rebuild
* make remove-devenv
* make run-devenv
* make start-devenv
* make stop-devenv

### make
Currently, the command generates `genimg/genimg`.
In the future, FAT32 raw disk image file of THEOS is generated by this command.
### make build-devenv
The command generates a docker image `theos-devenv`.
### make clean
Currently, the command removes `genimg/genimg` and intermediate files generated to build `genimg/genimg`
In the future, FAT32 raw disk image file of THEOS and intermediate files generated to build THEOS is removed by this command.
### make login-devenv
### make rebuild
Currently, the command regenerates `genimg/genimg`.
In the future, FAT32 raw disk image file of THEOS is regenerated by this command.
### make remove-devenv
The command remove a docker container `theos-devenv` and a docker image `theos-devenv`
### make run-devenv
### make start-devenv
### make stop-devenv
