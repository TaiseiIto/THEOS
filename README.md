# THEOS

My original operating system made with reference to [MikanOS](https://github.com/uchan-nos/mikanos/) by uchan-nos.

## thoes-devenv

To develop and use THEOS conveniently, I prepared [Dockerfile](/Dockerfile).
The docker image and container is built and run by the Dockerfile with `make devenv`.

## [Makefile](/Makefile)

The Makefile contains the following commands.

* make
* make clean
* make clean-devenv
* make devenv
* make rebuild
* make rebuild-devenv

### make
Currently, the command generates `genimg/genimg`.
In the future, FAT32 raw disk image file of THEOS is generated by this command.
### make clean
Currently, the command removes `genimg/genimg` and intermediate files generated to build `genimg/genimg`
In the future, FAT32 raw disk image file of THEOS and intermediate files generated to build THEOS is removed by this command.
### make clean-devenv
The command removes a docker container `theos-devenv` and a docker image `theos-devenv`
### make devenv
The command generates and logs in to a docker image `theos-devenv` and its container `theos-devenv`.
### make rebuild
Currently, the command removes and regenerates `genimg/genimg`.
In the future, FAT32 raw disk image file of THEOS is regenerated by this command.
### make rebuild-devenv
The command removes ,regenerates and logs in to a docker image `theos-devenv` and its container `theos-devenv`.
