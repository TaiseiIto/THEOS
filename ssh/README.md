# Files related to ssh 

There are 4 files, `README.md`, ssh `config` file, gitignored private key `github`, and `.gitignore` in this directory.
This directory is copied to `/~/ssh` when `../Dockerfile` is executed to build docker container `theos-devenv`.
Also, the directory copied to `/root/.ssh` when `../git/enpushable.sh` is executed by a developer to allow the executer to push commits to this repository.
